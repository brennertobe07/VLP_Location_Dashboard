📋 How to Use This Dashboard Each Election Cycle

1
Find the VA ELECT Early Voting Locations URL for the new election
Usually the same URL — elections.virginia.gov/casting-a-ballot/early-voting-office-locations/ — page content updates each cycle
2
Paste URL in the config field and click ⟳ Fetch
Claude fetches the live page and automatically extracts all satellite locations with schedules
3
Download the Polling Places XLSX from VA ELECT Registration Statistics page
New XLSX is posted for each election — download it to your computer first
4
Upload the XLSX using the file picker in the config strip
Election Day locations parse automatically — no server needed
5
Review the Overview tab for warnings
Schedules over 200 chars need trimming; missing VAN IDs need lookup in VAN
6
Use ↓ Export Excel buttons on Phoenix tabs
Downloads formatted XLSX ready to copy into the VLP submission spreadsheet for IWillVote upload
